# League of Legends API - Proof of Concept (PoC)

## Overview
This repository contains a short sprint project from 2024 focused on integrating and exploring the official Riot Games API. The primary goal of this Proof of Concept was to practice building a full-stack environment to handle asynchronous API calls, parse complex JSON payloads, and extract real-time player match data.

## Tech Stack
* **Frontend:** React
* **Backend:** Node.js
* **External Data:** Riot Games REST API

## Technical Focus
* **API Integration:** Establishing secure, real-time connections with the Riot Games API via a Node.js backend.
* **Frontend State Management:** Utilizing React to efficiently render UI components and display nested player statistics.
* **Asynchronous Logic:** Implementing async/await patterns to manage data load times and ensure smooth data handoffs between the backend and frontend.

## Project Status
**Archived / Learning Exercise.** This project successfully served its purpose as an architectural foundation for full-stack API data fetching and state management during my undergraduate studies. It is no longer being actively maintained.
